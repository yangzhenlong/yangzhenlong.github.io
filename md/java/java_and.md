# &和&&的区别

解释
```java
&和&&都是逻辑运算符，都是判断两边同时真则为真，否则为假；但是&&当第一个条件不成之后，后面的条件都不执行了，而&则还是继续执行，直到整个条件语句执行完为止。
```
举例说明
```java
public static void main(String[] args) {
    System.out.println("---------------测试&-----------------");
    int i=1;
    if(i++>3 & i++<9){
        System.out.println("abc-i");
    }
    System.out.println("使用&时，值加了2次，说明&后面的语句执行了。结果：i=" + i);

    System.out.println("---------------测试&&-----------------");
    int j=1;
    if(j++>3 && j++<9){
        System.out.println("abc-j");
    }
    System.out.println("使用&&时，值加了1次，说明&&后面的语句未执行。结果：j=" + j);
}
```
执行结果
```java
---------------测试&-----------------
使用&时，值加了2次，说明&后面的语句执行了。结果：i=3
```
```java
---------------测试&&-----------------
使用&&时，值加了1次，说明&&后面的语句未执行。结果：j=2
```
