Git怎么删除commit提交logs？

 2020-03-16   
 195 words    1 mins   
git git commit git log

我们从其他地方拉过来的代码，有以前别人提交过的记录日志

那我们想要清除这些日志该如何操作？请看下面

操作

创建临时分支

假设我们现在位于master分支，那么基于master分支创建一个临时分支temp

$ git checkout --orphan temp

添加所有要提交到文件

$ git add .

添加注释信息

$ git commit -m 'fix clean commit logs'

删除旧的master分支

$ git branch -D master

将当前临时分支更名为master

$ git branch -m master

推送代码到远程仓库

$ git push -f origin master

回顾

看了上面操作，通俗讲就是

  • 建一个没有任何 commit log 的临时分支
  • 然后删除老的 master 分支
  • 再将临时分支更名为 master
